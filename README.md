# Afrotech Water Solutions

Afrotech is a premier industrial water solutions provider based in Nairobi, Kenya. We specialize in high-performance pumping systems engineered for the demanding conditions of agriculture, industry, and municipal water supply across East Africa.

## 🚀 Project Overview

This repository contains the official Afrotech web platform, built with a premium "Midnight & Amber" industrial aesthetic and focused on technical clarity and high-conversion inquiry paths.

### Key Features
- **Dynamic Product Catalog**: A comprehensive technical database of submersible, solar, and centrifugal pumping systems.
- **Individual Spec Pages**: Deep-dive technical data for every pump model, including engineering specs and application guides.
- **Interactive Project Showcase**: High-impact case studies highlighting our impact on agricultural and industrial sectors.
- **Technical Inquiry Engine**: Optimized "Get A Quote" and contact systems for technical project requirements.
- **Premium Performance**: Optimized with Next.js 15 and GSAP for fluid, scroll-triggered animations.

## 🛠️ Technology Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + Custom CSS (globals.css)
- **Animations**: GSAP (GreenSock Animation Platform)
- **Icons**: @iconify/react
- **Fonts**: Playfair Display (Headings) & DM Sans (Body)
- **Maps**: Interactive Google Maps Embed

## 🏁 Development

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
```bash
git clone git@github.com:CodeWithFin/afrotech-solutions.git
cd afrotech-solutions
npm install
```

### Local Development
```bash
npm run dev
```
